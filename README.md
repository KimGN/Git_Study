## Git Branch

### git branch "branch name"
* 브랜치 생성

###git checkout "branch name"
* 브랜치 선택

###git checkout -b "branch name"
* 로컬 브렌치 생성
* -b 명령어 사용시 로컬 브렌치 생성, 체크아웃을 동시에 할수 있다

###git push origin "local branch name"
* remote branch 생성
